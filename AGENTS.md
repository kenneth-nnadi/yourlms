# Contributor notes

This repository is maintained as an open-source project on GitHub. See [CONTRIBUTING.md](CONTRIBUTING.md) for how to propose changes and [SECURITY.md](SECURITY.md) for reporting vulnerabilities.

When opening pull requests:

- Keep commits focused and descriptive.
- Run `php tests/run.php` and `php tests/integration_audit.php` when you change application code.
- Do not commit `config.local.php`, `uploads/`, database files, or large course export archives.
